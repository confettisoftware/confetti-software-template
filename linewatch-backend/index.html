Backend services for LineWatch+ Disney ride wait time notifications

Available Endpoints

POST /api/setup-database

Initialize database tables for device tokens, user alerts, and notification logs.

curl -X POST https://linewatch-backend.netlify.app/api/setup-database

POST /api/register-device

Register a device token for push notifications.

curl -X POST https://linewatch-backend.netlify.app/api/register-device -H "Content-Type: application/json" -d '{"deviceToken": "your_device_token"}'

GET /api/manage-alerts

Get all alerts for a device token.

curl -X GET https://linewatch-backend.netlify.app/api/manage-alerts -H "Content-Type: application/json" -d '{"deviceToken": "your_device_token"}'

POST /api/manage-alerts

Create a new alert for a ride.

curl -X POST https://linewatch-backend.netlify.app/api/manage-alerts -H "Content-Type: application/json" -d '{"deviceToken": "your_device_token", "parkId": "16", "parkName": "Disneyland", "rideId": "123", "rideName": "Space Mountain", "waitTimeThreshold": 30}'

DELETE /api/manage-alerts

Delete an alert by ID.

curl -X DELETE https://linewatch-backend.netlify.app/api/manage-alerts -H "Content-Type: application/json" -d '{"deviceToken": "your_device_token", "alertId": 1}'

GET /api/fetch-wait-times

Fetch current wait times for all Disney parks.

curl https://linewatch-backend.netlify.app/api/fetch-wait-times

POST /api/monitor-wait-times

Manual monitoring trigger (normally runs automatically every 5 minutes).

curl -X POST https://linewatch-backend.netlify.app/api/monitor-wait-times
