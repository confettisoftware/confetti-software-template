# App Store Review Update Note

We have updated our app to address the intellectual property concerns raised in the previous review:

## Legal Disclaimer Added

The app now includes the following disclaimer in both the App Store description and within the app itself:
"LineWatch+ is an independent app providing wait time data as a service to Disneyland and DCA visitors. Data provided by Queue-Times.com. This app is not affiliated with or endorsed by The Walt Disney Company."

## EULA Included

Apple's standard End User License Agreement has been added to the About section of the app, following the same implementation used by other approved theme park utility apps.

## Data Source Attribution

All wait time information is clearly attributed to Queue-Times.com, a third-party provider. Queue-Times.com handles all attraction naming with aliases removed to ensure generic, non-proprietary naming conventions. The app does not use any proprietary systems or data from theme parks.

## Updated App Store Description

The App Store description has been completely revised with the following improvements:

- Focused specifically on Disneyland and Disney California Adventure (DCA)
- Removed standalone "Disney" brand references to avoid trademark issues
- Used specific park names and land references (Adventureland, Tomorrowland, Cars Land, etc.)
- Streamlined language to be direct and functional
- Maintained professional tone while highlighting key features
- Updated keywords to be legally compliant

## Removal of Promotional Language

Any wording that could suggest endorsement, partnership, or official association with Disney or theme parks has been removed. The description now focuses purely on functionality and user benefits.

## Original and Generic Branding

The app uses neutral branding ("LineWatch+") and does not include any characters, logos, or copyrighted material belonging to any entertainment company.

## Trademark Compliance

- Removed standalone "Disney" references from keywords and description
- Focused on specific, legally safe terminology throughout

Our app functions strictly as a third-party utility, similar to other wait-time apps currently approved on the App Store. We believe the above revisions fully resolve the concerns raised and ensure full compliance with trademark guidelines.

Please let us know if there is anything else we should address — we're happy to make further adjustments if needed.
